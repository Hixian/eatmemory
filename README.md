#eatmemory

##1. Introduction
Simple utility to allocate memory on a computer

##2. What can I use this for?
- Test swap
- Test behaviors on a machine when there is little memory available

##3. Usage
```
eatmemory <size>
```

Size is in number of bytes, megabytes or gigabytes.

###Examples
```
eatmemory 1024
eatmemory 10M
eatmemory 4G
```
